AI

Asystent e-mailowy dla sklepu internetowego

Właściciel sklepu dostawał dziesiątki maili dziennie z pytaniami o zamówienia i dostępność. Zbudowałem system, który sam czyta te wiadomości, rozpoznaje o co chodzi i przygotowuje odpowiedzi. Do tego system przypomina właścicielowi o nieodpisanych mailach. Klienci dostają odpowiedzi szybciej, a sprzedawca nie siedzi nad skrzynką pół dnia.

LLM · Python · Integracja Gmail

Integracja KSeF dla biura rachunkowego

Biuro rachunkowe potrzebowało szybko podłączyć się do KSeF, zanim wejdą nowe przepisy. Podpiąłem ich system fakturowania tak, żeby faktury wysyłały się i pobierały automatycznie. Całość działała po 5 dniach roboczych.

KSeF API · Java · Spring Boot

CRM dla firmy usługowej
